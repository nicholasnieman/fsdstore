Redirect your website visitors based on geolocation (country, state, city, lat/lng/radius)

== Description ==

Create geo targeted url redirects for your website pages. Perfect for creating both simple and complex redirects.

This plugin can allow you to create simple redirects for your complete website or particular pages of your website. 

You can also use this plugin on multiple websites to inter-redirect between them which is especially useful if you own different country level domains.

This plugin is made by [Geolify](https://geolify.com "Geolify")


* Free version (with Geolify branding) & paid version (no branding) [see pricing](https://geolify.com/pricing "Pricing")

* High accuracy IP to location data

* No IP to location database installs required

**Features**

* Redirect visitors by country, state, city, IP address, latitude-longitude-radius

* Create multiple redirects each targeted at a particular geo location

* Option to add a default redirect for non-targeted visitors

* Option to restrict redirect to only certain pages

* Option to redirect all subpages of a particular page

* Option to redirect visitors coming from external sources only

* Option to bypass redirect if visitor comes from certain domains

* Option to pass query strings onto redirected url

* Option to pass sub directory path onto redirected url


**Detailed docs can be found below**

* [Geo Redirect](https://geolify.com/docs/geo-redirect/ "Geo Redirect")
* [Geo Redirect V2](https://geolify.com/docs/geo-redirect-v2/ "Geo Redirect V2")

The Geo Redirect V2 service is a more advanced version of the Geo Redirect service and is perfect for inter-redirecting between multiple websites.




== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `georedirect.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to https://geolify.com to create and account and setup your redirects
